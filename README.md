### pre-commit

Alternatively you can add the following repo to your `pre-commit-config.yaml`:

```yaml
  - repo: https://github.com/feluelle/airflint
    rev: v0.1.2-alpha
    hooks:
      - id: airflint
        args: ["-a"]  # Use -a for replacing inplace
```
